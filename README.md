# Summary
This project is a two-player chess game built using Python and Pygame, designed to provide a complete and interactive chess experience on a single system. It includes all standard chess rules such as castling, en passant, pawn promotion, and check detection, along with visual enhancements like move highlighting and board rotation.
The project was developed as a practical exercise to strengthen understanding of game development concepts, including state management, coordinate handling, and user interaction. AI tools were used during development to assist with problem-solving and structuring, while ensuring a clear understanding of the underlying logic and implementation.
Overall, this project reflects the ability to build a well-structured, feature-rich application with clean logic and an engaging user interface.

 # Features
 Full chess gameplay (2-player)
 Turn-based system
 Board rotation after each move
 Smooth animations
 Valid move highlighting
 Capture tracking
 Castling
 En passant
 Pawn promotion
 Check detection
